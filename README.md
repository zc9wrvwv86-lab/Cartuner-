# CarTuner Mod Planner

A simple static MVP for a car tuning recommendation website.

## Features

- Enter make, model, engine, current HP, and target HP
- Calculates estimated tuning stage
- Shows recommended supporting modifications
- Includes known engine presets
- Works as a static website
- Ready for GitHub Pages

## Files

- `index.html`
- `styles.css`
- `script.js`

## Run locally

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Go to repository Settings
2. Open Pages
3. Source: Deploy from branch
4. Branch: main
5. Folder: `/root`
6. Save

Your site will be live after GitHub builds it.
